# pangea-azure

Azure provider resources for the Pangea infrastructure DSL. All resource
code is **auto-generated** by `pangea-forge` via `iac-forge-cli`. Manual edits
to resource files will be overwritten on the next generation run.

## Structure

```
lib/
  pangea-azure.rb                     # Entry point (generated -- requires all resources)
  pangea-azure/version.rb             # Version constant (manual)
  pangea/
    resources/
      types/azure/
        core.rb                       # Provider-specific Dry::Types (generated)
      azure.rb                        # Aggregator module (generated)
      azurerm_*/                      # 1,124 resource directories (generated)
        types.rb                      # Dry::Struct attributes class
        resource.rb                   # ResourceBuilder DSL
spec/
  spec_helper.rb                      # Test setup (manual)
  support/                            # Shared test helpers
  resources/                          # RSpec stubs (generated)
```

## Generated vs Manual

| File | Source |
|------|--------|
| `lib/pangea/resources/azurerm_*/` | Generated by pangea-forge |
| `lib/pangea-azure.rb` | Generated (require file) |
| `lib/pangea/resources/azure.rb` | Generated (aggregator) |
| `lib/pangea/resources/types/azure/` | Generated (type definitions) |
| `spec/resources/` | Generated (RSpec stubs) |
| `lib/pangea-azure/version.rb` | Manual |
| `spec/spec_helper.rb` | Manual |
| `Gemfile`, `*.gemspec`, `flake.nix` | Manual |

## Resource count

- **1,124 resources** — full coverage of the `azurerm` Terraform provider,
  generated by `terraform-schema-importer`

## Provider module

- Module: `Pangea::Resources::Azure`
- Resource prefix: `azurerm_`
- Include: `include Pangea::Resources::Azure`

## Dependencies

- pangea-core ~> 0.2
- terraform-synthesizer ~> 0.0.28
- dry-types ~> 1.7
- dry-struct ~> 1.6

## Testing

```sh
bundle exec rspec                            # Run all tests
bundle exec rspec spec/resources/            # All resource specs
```

## Regeneration

Resources are generated by `pangea-forge` (the Pangea backend of `iac-forge`).
Run regeneration via `iac-forge-cli`:

```sh
terraform-schema-importer generate \
  --schema schema.json \
  --provider "registry.terraform.io/hashicorp/azurerm" \
  --provider-name azurerm \
  --output-dir .
```
